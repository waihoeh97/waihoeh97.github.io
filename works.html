Random Shooter

This is a hypercasual game made using Unity for Android

A hypercasual game where unlike the usual ones, contains very simple procedural level generation with gameplay of auto-shooting and scoring of shooting enemies.

This game uses procedural generation for its level and its generation can be seen when the players press the restart button to regenerate the level.
The level currently only supports simple generation of perfect boxes (same sides) with opening of different sides. Careful consideration is taken to prevent the overlapping of boxes and also preventing the boxes from generating outside designated area.

List of mechanics
- Procedural Level Generation
- Object Pooling for enemies
- Player auto shooting
- Player with on screen controls for Android / mobile devices
- Score system

This Game may be further developed to improve playability and more extensive features in the future

CookBeat

Promotional Video

This game is made using Unity

Made with a team of artists and programmers
A 3D rhythm simulation game where the player has to cook according to the beat.

CookBeat is an Android rhythm cooking game where the genre of rhythm games and cooking games is mixed. The game is heavily based on Rhythm Heaven and Cooking Mama. CookBeat’s rhythm is not the typical note system rhythm game but relies on one’s hearing and aptitude to music, much like Rhythm Heaven. Lots of research has been done to make the game as fun as possible despite having some of our members who has no musical knowledge whatsoever. The theme is centralized to Italian recipes and music. So everything in the game would be Italian style. The gameplay is very linear and references heavy from Rhythm Heaven’s gameplay.

Tasked of creating the song manager.
It is a system where it can accept player inputs and detects whether it syncs with the rhythm. It acts as a notifier to many other elements like the animations, the UI and the scores
